# EduPro_Project_variations
# 🎓 EduPro Instructor Performance & Course Quality Evaluation

## 📌 Overview
This project analyzes instructor performance and course quality using user data. Since instructor-specific data was not available, synthetic features such as experience, expertise, and ratings were generated.

## 📊 Features
- Experience vs Teacher Rating analysis
- Teacher Rating vs Course Rating comparison
- Expertise-based performance
- Top instructors identification
- Interactive Streamlit dashboard

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Streamlit

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run preprocessing
python preprocess.py

### 3. Run EDA
python eda.py

### 4. Run dashboard
python -m streamlit run app/streamlit_app.py

## 📈 Output
- EDA visualizations saved as images
- Interactive dashboard with KPI metrics and charts
